Git Commands

git init - initializes an empty git repository in a directory

git status - displays any files that have been changed since the last commit

git add - adds specified files to the staging area

git commit - saves staged files to local repository with specified commit message

git log - displays a summary of all past commits

git push - sends contents of local repository to remote repository

git pull - merges any changes in remote repository to local repository

git clone - makes a copy of an existing remote repository on local machine

git merge - merges code of specified branch with current branch

git branch - displays a list of all existing branches OR create a new branch

git branch --delete - deletes a specified branch

git checkout - changes working directory to a different branch OR a previous commit
